# Images Folder

This folder contains all images for the Asker Skadedyrkontroll website.

## Required Images

### Logo

- `logo.png` or `logo.svg` - Company logo

### Service Icons

- `service-icon-1.svg` - Boliginspeksjon icon
- `service-icon-2.svg` - Miljøvennlig Behandling icon
- `service-icon-3.svg` - Oppfølging icon

### Content Images

- `about-placeholder.jpg` - About section image
- `hero-background.jpg` - Hero section background (optional)

## Image Guidelines

- Use PNG or SVG format for logos and icons
- Use JPG format for photos
- Optimize images for web (compress without losing quality)
- Recommended sizes:
  - Logo: 200x60px
  - Service icons: 80x80px
  - About image: 600x400px
  - Hero background: 1920x1080px

## Placeholder Images

Until you have the actual images, you can use placeholder services like:

- https://placeholder.com/
- https://picsum.photos/
- https://unsplash.com/

Example: `https://picsum.photos/600/400` for the about image
